Running the file MS-Apriori.py

1. The input and parameter files should be present in the same folder as the python file( “MS-Apriori.py <input-filename> <parameter-filename> <output-filename>”)
2. In order for the code to run successfully, 3 inputs are required: the input file, the parameter file and the output file. 
3. After execution, the output is printed in the output file and output file is stored in the same directory.
